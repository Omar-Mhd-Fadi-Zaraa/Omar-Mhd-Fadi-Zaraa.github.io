# Portfolio — Up-and-Coming ML Engineer

This repository contains a simple, responsive single-file portfolio landing page for an up-and-coming Machine Learning engineer.

Files:
- `index.html` — main landing page (links to `styles.css`).
- `styles.css` — extracted styles for the page.
- `CNAME` — contains the configured Pages domain (if applicable).

Preview locally:

```bash
# from the repository root
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

To publish with GitHub Pages, push this branch to `master` on the `Omar-Mhd-Fadi-Zaraa.github.io` repository (the repo is already named for GitHub Pages). If you have a custom domain, replace the contents of `CNAME` with your domain.

Replace placeholders in `index.html` with your real name, contact email, and project links.
